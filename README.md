# YouTube 데이터셋 샘플

<h2>1001개의 YouTube 댓글로 구성된 샘플 데이터셋</h2>

![YouTube dataset header](https://github.com/bright-kr/YouTube-dataset-sample/blob/main/YouTube-dataset.png)

1000개 이상의 레코드로 구성된 YouTube 데이터셋 샘플입니다. 데이터셋은 <b>Bright Data API</b>를 사용하여 추출되었습니다.

<h2>YouTube 데이터셋에 포함된 일부 데이터 포인트:</h2>

* ```comment_id```: 각 댓글의 고유 식별자  
* ```comment_text```: 댓글의 텍스트 콘텐츠  
* ```likes```: 댓글이 받은 좋아요 수  
* ```replies```: 댓글에 대한 답글 수  
* ```username```: 댓글 작성자의 사용자 이름  
* ```username_md5```: username 필드의 MD5 해시  
* ```user_channel```: 댓글 작성자의 YouTube 채널 URL  
* ```date```: 댓글이 게시된 날짜  
* ```url```: 댓글이 게시된 YouTube 동영상의 웹 주소  
* ```video_id```: YouTube 동영상의 고유 식별자  
* ```replies_value```: 답글 값 배열  
* ```replies_without_names```: 사용자 이름이 연결되지 않은 답글

그 외에도 더 많은 항목이 있습니다.

이는 "YouTube comments" 데이터셋에서 파생된 샘플 부분집합이며,
해당 데이터셋에는 <b>11.1M records</b> 이상이 포함되어 있습니다.

<b>기타 제공되는 YouTube 데이터셋:</b>

* <b>YouTube posts dataset</b> — <b>1.3B records</b> 포함

* <b>YouTube profiles dataset</b> — <b>29.6M records</b> 포함 

사용 가능한 데이터셋 파일 형식: <b>JSON, NDJSON, JSON Lines, CSV, or Parquet. Optionally, files can be compressed to .gz</b>.

데이터셋 전달 방식 옵션: <b>Email, API download, Webhook, Amazon S3, Google Cloud storage, Google Cloud PubSub, Microsoft Azure, Snowflake, SFTP</b>.

업데이트 빈도: <b>Once, Daily, Weekly, Monthly, Quarterly, or Custom basis</b>.

추출된 데이터 포인트에 추가로 제공 가능한 데이터 강화: <b>요청 기반입니다.</b>

<b>[전체 YouTube 데이터셋 받기](https://brightdata.co.kr/products/datasets/youtube)</b>.

<h2>YouTube 데이터셋의 활용 사례는 무엇입니까?</h2>

<h3>1. 소비자 감성</h3>
YouTube 구독자들이 귀사의 브랜드에 대해 무엇을 말하고 있는지에 대한 인사이트를 확보합니다. YouTube에서 댓글, 해시태그, 멘션, 좋아요를 수집하여 브랜드 및 제품 멘션을 모니터링함으로써 브랜드 평판을 강화하고 서비스를 개선할 수 있습니다.

<h3>2. 콘텐츠 제작</h3>
YouTube 데이터셋을 활용하여 트렌딩 동영상, 인기 형식, 사용자가 가장 많이 참여하는 콘텐츠에 대한 인사이트를 발굴합니다. 이 데이터를 활용해 최대 도달 범위와 시청자 참여를 위해 동영상 콘텐츠를 최적화할 수 있습니다.

<h3>3. 시장 분석</h3>
YouTube 데이터셋으로 경쟁사의 YouTube 내 존재감을 분석합니다. 경쟁사의 콘텐츠와 활동에 대한 데이터를 수집하고, 참여를 유도하는 콘텐츠 유형을 파악하며, YouTuber들이 해당 브랜드와 서비스에 대해 무엇을 말하는지 확인할 수 있습니다.
